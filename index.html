Projects

Twist The Web Clone

Live Demo Repository
  • A web based social speedsolving website based on the now defunct Twist The Web.
  • Allows users to join lobbies in which all users in a given lobby are given a scramble on the Rubiks Cube.
  • Users can time their solves and the solves are sent back to the lobby and shared with users.
  • Each lobby includes a chatroom for which users can discuss their solves in.
  • Backend built in Python with Flask-Socketio and using MongoDB. Frontend built in Typescript with React.

Huffman Encoding

Live Demo Respository
  • A web based visualisation tool for the Huffman Encoding algorithm.
  • Accepts user input for the probability of given symbols and generates the Huffman Tree formed and the encoding of the symbols.
  • A specific encoding can be selected and the traceback of the encoding path drawn.
  • Built in Javascript with the use of HTML5 Canvas for visuals.

  • Built in Javascript with the use of HTML5 Canvas for visuals.
