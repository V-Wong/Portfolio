Projects

Twist The Web Clone

Live Demo
  • A web based social speedsolving website based on the now defunct Twist The Web.
  • Allows users to join lobbies in which all users in a given lobby are given a scramble on the Rubiks Cube.
  • Users can time their solves and the solves are sent back to the lobby and shared with users.
  • Each lobby includes a chatroom for which users can discuss their solves in.
